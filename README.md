# Inventory Management System

## Tech Stack
- Java
- Spring Boot
- Spring Security
- JWT Authentication
- Microservices
- API Gateway
- Eureka Server
- MySQL
- ReactJS
- Bootstrap / Reactstrap

## Features
- User Authentication & Authorization
- Product Management
- Order Management
- Inventory Tracking
- JWT Based Security
- API Gateway Routing
- Service Discovery
- Role Based Access

## Microservices
- AuthenticationService
- UserService
- ProductService
- InventoryService
- OrderService
- AlertService
- Gateway
- ServiceRegistry

## Frontend
React based UI for:
- Add Product
- Update Product
- Delete Product
- View Products
- Authentication

## Run Project
### Backend
Run all Spring Boot services individually.

### Frontend
```bash
npm install
npm start
```
